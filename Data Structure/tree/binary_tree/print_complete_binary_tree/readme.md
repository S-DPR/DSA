1-Based 트리배열을 입력받아 완전이진트리의 형태로 출력하는 파이썬 함수입니다.
Bottom-Up 세그먼트 트리를 예시로 사용하였으며, 아래와 같이 출력됩니다.
-------------------------------------------------------------
                               15
              10                              05
      09              01              02              03
  04      05
-------------------------------------------------------------
배열 내에서 가장 큰 수의 길이에 맞춰 출력되는 길이가 정해집니다.
시간복잡도는 O(N)입니다.
